## v0.1.0

这是 SyncLan 的首个 **Release Candidate (RC)** 版本，也是正式版发布前的候选版本。

欢迎大家下载安装体验，并帮助测试功能与稳定性。如果没有发现阻碍发布的问题，该版本将作为 **v0.1.0** 正式版发布。

### ✨ 当前已支持

- 💬 局域网即时聊天
- 🖼️ 图片发送与预览
- 🔍 自动发现局域网设备
- 🌐 浏览器访问（Web）
- 🖥️ Windows / macOS / Linux 桌面客户端
- 🔄 自动更新（正式版支持）

### ⚠️ 注意

- 这是 **预发布版本（Pre-release）**。
- 功能和界面仍可能在正式版发布前进行调整。
- 欢迎通过 GitHub Issues 提交 Bug 报告和功能建议。

感谢所有参与测试和提供反馈的用户 ❤️
