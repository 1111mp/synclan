## v0.1.0

这是 SyncLan 的首个 **Release Candidate (RC)** 版本，也是正式版发布前的候选版本。

欢迎大家下载安装体验，并帮助测试功能完整性与稳定性。如果没有发现影响发布的问题，该版本将作为 **v0.1.0 正式版** 发布。

### ✨ 已支持功能

- 💬 局域网即时聊天
- 📁 文件传输
- 🖼️ 图片发送与预览
- 🎬 视频发送与预览
- 🔍 自动发现局域网设备
- 🌐 浏览器访问（Web）
- 🖥️ Windows / macOS / Linux 桌面客户端
- 🔄 自动更新

### ⚠️ 注意

- 当前版本为 **预发布版本（Pre-release）**
- 功能和界面仍可能在正式版发布前调整
- 欢迎通过 GitHub Issues 提交 Bug 和功能建议

感谢所有参与测试和提供反馈的用户 ❤️
